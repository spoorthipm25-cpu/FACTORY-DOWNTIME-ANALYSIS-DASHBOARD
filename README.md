# 🏭 Factory Downtime Analysis Dashboard

## 📌 Project Overview
This project analyzes factory downtime data using Tableau to identify key factors affecting operational efficiency.

## 👩‍💻 Role
Data Analyst Intern – Deloitte Virtual Internship

## 🎯 Objective
- Identify factories contributing to maximum downtime
- Analyze machine-level breakdown patterns
- Build an interactive dashboard for insights

## 📊 Tools Used
- Tableau
- Data Analysis
- Dashboard Design

## 🔍 Key Insights
- Factory Seiko contributes ~38% of total downtime
- Osaka, Japan recorded the highest breakdown frequency
- Laser Welding & Laser Cutting machines caused maximum downtime
- Berlin factory showed highest efficiency

## 📈 Recommendations
- Focus maintenance on high-impact machines
- Use Berlin factory as a benchmark
- Monitor Shenzhen factory to prevent decline
